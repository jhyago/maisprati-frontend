FIFO — First In, First Out

Filas (Queue)

O primeiro a entrar é o primeiro a sair — como uma fila de banco. Entra no fim (enqueue), sai pela frente (dequeue).

01

Conceito

Entra atrás, sai na frente

Usada em: filas de impressão, processamento de mensagens, busca em largura (BFS), agendamento de tarefas. Mantendo ponteiros para front e rear, ambas as operações são O(1).

O(1) ✓

Experimente: enfileire (enqueue) novos itens no fim e remova (dequeue) pela frente. Veja os rótulos front e rear se moverem.

fila.js 
// FILA com LinkedList: ponteiros front e rear
//   front → quem sai a seguir | rear → onde entram novos
// Sem rear, cada enqueue seria O(n).
class Node {
  constructor(v) { this.valor = v; this.proximo = null; }
}

class Queue {
  constructor() { this.front = null; this.rear = null; this.size = 0; }

  enqueue(v) {        // entra no fim — O(1)
    const n = new Node(v);
    if (!this.rear) { this.front = n; this.rear = n; }
    else { this.rear.proximo = n; this.rear = n; }
    this.size++;
  }

  dequeue() {        // sai pela frente — O(1)
    if (!this.front) return null;
    const v = this.front.valor;
    this.front = this.front.proximo;
    if (!this.front) this.rear = null;
    this.size--;
    return v;
  }

  peek() { return this.front ? this.front.valor : null; }
}

const fila = new Queue();
fila.enqueue(10); fila.enqueue(20); fila.enqueue(30);
console.log("peek:", fila.peek());       // 10
console.log("dequeue:", fila.dequeue()); // 10
console.log("dequeue:", fila.dequeue()); // 20
⚠ Cuidado com Queue só com Array É tentador usar arr.push() e arr.shift(), mas shift() é O(n) (desloca todos). Para filas grandes, prefira LinkedList ou um índice de cabeça.

ASimulação de Atendimento ◆ Médio

Processe a fila de atendimento completamente. Para cada cliente, imprima quando começou e terminou (em minutos), e ao final o tempo total.

Seu código
solução_A.js
let relogio = 0;
while (fila.length > 0) {
  const cliente = fila.shift(); // dequeue
  const inicio = relogio;
  relogio += cliente.tempo;
  console.log(`${cliente.nome.padEnd(6)} ${inicio}min → ${relogio}min`);
}
console.log(`Tempo total: ${relogio} minutos`);
BBatata Quente (Hot Potato) ◆ Médio

Simule o jogo da batata quente: uma fila circular onde, a cada rodada, os primeiros k-1 participantes são reenfileirados e o k-ésimo é eliminado. O último restante vence.

  • Use .push() e .shift() para simular a fila circular
  • A cada rodada: passe k-1 participantes para o fim da fila, elimine o k-ésimo
  • Repita até restar apenas um nome
Seu código
solução_B.js
function batataquente(nomes, k) {
  const fila = [...nomes];
  while (fila.length > 1) {
    for (let i = 0; i < k - 1; i++) {
      fila.push(fila.shift()); // passa a batata
    }
    const eliminado = fila.shift();
    console.log(`Eliminado: ${eliminado}`);
  }
  return fila[0];
}
CNúmeros Binários com Fila ◆ Médio

Use uma fila para gerar os n primeiros números em binário. Esta é uma aplicação clássica de BFS implícito: cada número gera seus dois "filhos" adicionando "0" e "1" ao fim.

  • Comece com "1" na fila
  • A cada iteração: desenfileire o atual, adicione ao resultado, enfileire atual+"0" e atual+"1"
  • Repita n vezes
Seu código
solução_C.js
function binariosComFila(n) {
  const fila = ["1"];
  const resultado = [];
  while (resultado.length < n) {
    const atual = fila.shift();
    resultado.push(atual);
    fila.push(atual + "0");
    fila.push(atual + "1");
  }
  return resultado;
}
