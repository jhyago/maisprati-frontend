Nós conectados por ponteiros

Lista Simplesmente
Encadeada

Nós espalhados pela memória, cada um guardando um valor e um ponteiro para o próximo. O último aponta para null, sinalizando o fim da corrente.

01

Conceito

head → ... → null

Diferente do array, os nós não ficam em posições contíguas. Por isso não há acesso por índice em O(1): para chegar ao 3º nó, é preciso seguir os ponteiros desde o head. Em compensação, inserir no início é O(1) — basta criar o nó e apontar.

Experimente: insira e remova nós e veja os ponteiros se religando. Note como inserir no início é instantâneo, mas inserir no fim precisa percorrer a corrente.

lista_simples.js 
class Node {
  constructor(valor) {
    this.valor = valor;   // dado
    this.proximo = null;  // ponteiro p/ o próximo
  }
}

class LinkedList {
  constructor() { this.head = null; this.length = 0; }

  // O(1) — só troca ponteiros
  inserirInicio(valor) {
    const novoNo = new Node(valor);
    novoNo.proximo = this.head;
    this.head = novoNo;
    this.length++;
  }

  // O(n) — percorre até o último nó
  inserirFim(valor) {
    const novoNo = new Node(valor);
    if (!this.head) { this.head = novoNo; }
    else {
      let atual = this.head;
      while (atual.proximo) atual = atual.proximo;
      atual.proximo = novoNo;
    }
    this.length++;
  }

  imprimir() {
    let atual = this.head, res = "";
    while (atual) { res += atual.valor + " → "; atual = atual.proximo; }
    console.log(res + "null");
  }
}

const lista = new LinkedList();
lista.inserirInicio(10);
lista.inserirInicio(20);
lista.inserirInicio(30);
lista.imprimir();      // 30 → 20 → 10 → null
lista.inserirFim(5);
lista.imprimir();      // 30 → 20 → 10 → 5 → null

🔑 Por que inserir no início é O(1)? Não há percurso — três operações: criar o nó, apontar seu proximo para o head atual, e mover o head. Independe do tamanho. Num array, isso seria O(n) (deslocaria todos os elementos).

Exercício

AInverter a Lista ◆ Médio

Implemente inverter() que reverte a ordem dos nós sem criar nós novos — apenas re-religando os ponteiros.

  • Antes: 10 → 20 → 30 → null
  • Depois: 30 → 20 → 10 → null
Seu código
solução_A.js
inverter() {
  let anterior = null;
  let atual = this.head;
  while (atual) {
    const proximo = atual.proximo; // guarda próximo
    atual.proximo = anterior;       // inverte ponteiro
    anterior = atual;                // avança anterior
    atual = proximo;                  // avança atual
  }
  this.head = anterior;
}
