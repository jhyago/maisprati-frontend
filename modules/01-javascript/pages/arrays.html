Sequências indexadas

Arrays, for...of
& forEach

Arrays guardam valores em sequência, acessados por índice. E os laços feitos para percorrê-los (e a outros iteráveis) são o for...of e o forEach — que entregam o valor diretamente.

01

Arrays

Memória contígua

Um array é uma sequência de elementos em posições contíguas de memória. Cada elemento tem um índice numérico (começando em 0), o que permite acesso direto em O(1) — para chegar na posição 5, basta arr[5], sem percorrer nada.

Array de 5 elementos — memória contígua
10
25
42
7
88
Acesso: arr[3] = 7 (instantâneo, sem percurso).
Acesso por índice
O(1) ✓
Push (fim)
O(1) amortizado
Shift (início)
O(n)
Busca por valor
O(n)
01_arrays_basico.js 
const numeros = [10, 25, 42, 7, 88];

// Acesso por índice — O(1)
console.log("Primeiro:", numeros[0]);
console.log("Último:", numeros[numeros.length - 1]);

// Métodos que MODIFICAM o array
numeros.push(99);     // fim    → O(1)
numeros.unshift(5);   // início → O(n) (desloca tudo)

// Métodos que retornam NOVO array
const dobrados = numeros.map(n => n * 2);
const pares = numeros.filter(n => n % 2 === 0);
const soma = numeros.reduce((acc, n) => acc + n, 0);

console.log("Array:", numeros);
console.log("Dobrados:", dobrados);
console.log("Pares:", pares);
console.log("Soma:", soma);
💡 Por que push/pop são O(1) mas shift/unshift são O(n)? Adicionar no fim só ocupa a próxima posição livre. Mas adicionar no início obriga o JS a deslocar TODOS os elementos uma posição — quanto mais elementos, mais lento.
02

for...of

Iteráveis → Valores

O for...of itera sobre os valores de qualquer iterável: arrays, strings, Sets, Maps. Diferente do for...in (que dá as chaves), o for...of entrega diretamente cada valor — e aceita break/continue.

02_for_of_arrays.js 
// FOR...OF com ARRAYS — a variável recebe CADA VALOR
const frutas = ["maçã", "banana", "laranja"];
for (const fruta of frutas) console.log("Fruta:", fruta);

console.log("---");

// Com índice via entries() → [índice, valor]
for (const [i, fruta] of frutas.entries()) {
  console.log(`[${i}] → ${fruta}`);
}

console.log("---");

// Array de objetos com break
const alunos = [
  { nome: "Carlos", nota: 8.5 },
  { nome: "Beatriz", nota: 9.2 },
  { nome: "Diego", nota: 5.0 },
];
for (const aluno of alunos) {
  if (aluno.nota < 6) { console.log("Parou em reprovado:", aluno.nome); break; }
  console.log(aluno.nome, "aprovado");
}
03_for_of_outros.js 
// FOR...OF funciona com QUALQUER iterável

// String → caractere por caractere
let juntas = "";
for (const letra of "Código") juntas += letra + "-";
console.log("Letras:", juntas);

// Set → valores únicos (ignora duplicatas)
const ids = new Set([1, 2, 2, 3, 3]);
for (const id of ids) console.log("ID:", id);

// Map → pares chave/valor preservando a ordem
const capitais = new Map([["Brasil", "Brasília"], ["Japão", "Tóquio"]]);
for (const [pais, capital] of capitais) console.log(`${pais} → ${capital}`);
🚫 for...of NÃO funciona com objetos simples! Objetos { } não são iteráveis. for (x of { a: 1 }) lança TypeError. Para iterar objetos, use for...in (página de Objetos) ou converta com Object.entries(obj) — exemplo abaixo.
04_for_of_objeto.js 
// for...of + Object.entries() → iterar chave+valor de objetos
const precos = { arroz: 8.50, feijao: 7.20, azeite: 22.90 };

let total = 0;
for (const [produto, preco] of Object.entries(precos)) {
  total += preco;
  console.log(`${produto.padEnd(8)}: R$ ${preco.toFixed(2)}`);
}
console.log(`Total: R$ ${total.toFixed(2)}`);
03

forEach

Arrays → Callback

O forEach é um método de array que executa uma função (callback) para cada elemento. É expressivo e legível, ideal para efeitos colaterais (imprimir, acumular, atualizar).

⚠ forEach não para no meio — e não tem retorno! Diferente de for...of, o forEach não aceita break. Se precisar interromper, use for...of. E para transformar dados, prefira .map() (forEach sempre retorna undefined).
05_foreach.js 
// forEach((elemento, índice, array) => { ... })
const numeros = [10, 20, 30, 40];

numeros.forEach((num, idx, arr) => {
  const ultimo = idx === arr.length - 1;
  console.log(`[${idx}] ${num}${ultimo ? " ← último" : ""}`);
});

// Acumulando em variável externa
let soma = 0;
numeros.forEach(n => { soma += n; });
console.log("Soma:", soma);
06_foreach_objetos.js 
// forEach com array de objetos — caso real (nota fiscal)
const pedidos = [
  { produto: "Teclado", preco: 350, qtd: 2 },
  { produto: "Mouse", preco: 120, qtd: 1 },
  { produto: "Monitor", preco: 1800, qtd: 1 },
];

let total = 0;
pedidos.forEach(({ produto, preco, qtd }) => {
  const sub = preco * qtd;
  total += sub;
  console.log(`${produto.padEnd(10)} x${qtd} = R$ ${sub.toFixed(2)}`);
});
console.log(`TOTAL: R$ ${total.toFixed(2)}`);
04

Comparativo de Laços

Quando usar cada um
LaçoItera sobreRecebebreak/continueQuando usar
for...in Objetos Chaves (strings) ✓ sim Iterar propriedades de um objeto
for...of Iteráveis (array, string, Set, Map) Valores ✓ sim Arrays quando precisa de break/await
forEach Arrays Valor + Índice + Array ✗ não Efeito colateral para cada item, sem parar
for clássico Qualquer Índice ✓ sim Controle fino de índices, passos, loops inversos
07_comparativo.js 
// Mesmo dado, laços diferentes
const notas = { maria: 9, joao: 7, ana: 6 };
const array = [9, 7, 6];

console.log("─ for...in (chaves do objeto) ─");
for (const aluno in notas) console.log(aluno, "→", notas[aluno]);

console.log("─ for...of + entries (objeto) ─");
for (const [aluno, nota] of Object.entries(notas)) console.log(aluno, "→", nota);

console.log("─ forEach (array) ─");
array.forEach((nota, i) => console.log(`[${i}] ${nota}`));

Exercícios

Pratique aqui
AEstatísticas de um Array ● Fácil

Dado o array de números, calcule a soma, a média, o maior e o menor valor, e um novo array só com os valores acima da média.

Seu código
solução_A.js
const soma = numeros.reduce((a, n) => a + n, 0);
const media = soma / numeros.length;
console.log("Soma:", soma);
console.log(`Média: ${media.toFixed(2)}`);
console.log("Maior:", Math.max(...numeros), "| Menor:", Math.min(...numeros));
console.log("Acima da média:", numeros.filter(n => n > media));
BTemperaturas com for...of ● Fácil

Use for...of para calcular a soma, a média, a máxima e a mínima das temperaturas.

Seu código
solução_B.js
let soma = 0, max = -Infinity, min = Infinity;
for (const t of temperaturas) {
  soma += t;
  if (t > max) max = t;
  if (t < min) min = t;
}
console.log(`Soma: ${soma.toFixed(1)} | Média: ${(soma/temperaturas.length).toFixed(1)}`);
console.log(`Máxima: ${max} | Mínima: ${min}`);
CFolha de Pagamento com forEach ◆ Médio

Use forEach para imprimir cada funcionário, dar 10% de aumento a quem ganha menos de R$ 3.000, e calcular a folha total.

Seu código
solução_C.js
let folha = 0;
funcionarios.forEach(f => {
  if (f.salario < 3000) f.salario *= 1.1;
  folha += f.salario;
  console.log(`${f.nome.padEnd(8)}: R$ ${f.salario.toFixed(2)}`);
});
console.log(`Folha total: R$ ${folha.toFixed(2)}`);
