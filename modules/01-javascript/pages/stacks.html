LIFO — Last In, First Out

Pilhas (Stack)

O último elemento inserido é o primeiro a sair — como uma pilha de pratos. Operações no topo: push, pop e peek, todas em O(1).

01

Conceito

Tudo no topo

Usada em: Ctrl+Z (desfazer), a call stack de funções, verificação de parênteses, navegação voltar/avançar. Só se mexe no topo — por isso todas as operações são O(1).

push (topo)
O(1) ✓
pop (topo)
O(1) ✓
peek
O(1) ✓
Buscar elemento
O(n)

Experimente: empilhe (push) e desempilhe (pop). O topo é sempre a última peça colocada — só ela entra e sai. Use peek para espiar sem remover.

pilha.js 
// PILHA com array: o fim do array é o TOPO
class Stack {
  constructor() { this.items = []; }
  push(v) { this.items.push(v); }            // O(1)
  pop() { return this.items.pop() ?? null; } // O(1)
  peek() { return this.items[this.items.length - 1] ?? null; }
  tamanho() { return this.items.length; }
}

// Uso clássico: parênteses balanceados
function balanceados(expr) {
  const p = new Stack();
  for (const ch of expr) {
    if (ch === "(") p.push(ch);
    else if (ch === ")") {
      if (p.tamanho() === 0) return false;
      p.pop();
    }
  }
  return p.tamanho() === 0;
}

const p = new Stack();
p.push(10); p.push(20); p.push(30);
console.log("peek:", p.peek()); // 30
console.log("pop:", p.pop());   // 30
console.log("((a+b)):", balanceados("((a+b))")); // true
console.log("((a+b):", balanceados("((a+b)"));   // false
🔑 Array ou LinkedList para pilha? Em JavaScript, o Array nativo costuma ser mais rápido (cache-friendly, sem overhead de ponteiros). A LinkedList brilha quando se quer O(1) garantido sem custo amortizado de redimensionamento.

APalíndromo com Pilha ◆ Médio

Use uma pilha para verificar se uma palavra é palíndromo (igual lida de trás para frente).

  • Empilhe cada caractere
  • Desempilhe comparando com a posição correspondente
Seu código
solução_A.js
function ehPalindromo(palavra) {
  const pilha = [];
  for (const ch of palavra) pilha.push(ch);
  for (let i = 0; i < palavra.length; i++) {
    if (palavra[i] !== pilha.pop()) return false;
  }
  return true;
}
BInverter Array com Pilha ◆ Fácil

Use uma pilha para inverter um array sem usar .reverse(). A ideia: empilhe todos os elementos e desempilhe em um novo array.

  • Empilhe cada item do array original
  • Desempilhe tudo em um array resultado
  • Por que isso inverte? Porque a pilha é LIFO
Seu código
solução_B.js
function inverter(arr) {
  const pilha = [];
  const resultado = [];
  for (const item of arr) pilha.push(item);
  while (pilha.length > 0) resultado.push(pilha.pop());
  return resultado;
}
CMinStack — Mínimo em O(1) ◆ Difícil

Implemente uma pilha que, além de push/pop/peek, oferece um método min() que retorna o menor valor atual em O(1).

  • A pilha auxiliar mins guarda o mínimo acumulado até cada ponto
  • Quando empilhar v: o novo mínimo é Math.min(v, mins.peek())
  • Quando desempilhar: remova de ambas as pilhas simultaneamente
Seu código
solução_C.js
class MinStack {
  constructor() { this.main = []; this.mins = []; }

  push(v) {
    this.main.push(v);
    const atual = this.mins.length === 0
      ? v
      : Math.min(v, this.mins[this.mins.length - 1]);
    this.mins.push(atual);
  }

  pop() { this.mins.pop(); return this.main.pop() ?? null; }
  peek() { return this.main[this.main.length - 1] ?? null; }
  min() { return this.mins[this.mins.length - 1] ?? null; }
}
