Coleções chave→valor

Objetos & for...in

O objeto é a estrutura mais versátil do JavaScript. E como ele guarda pares chave: valor, o laço feito sob medida para percorrê-lo é o for...in, que itera sobre as chaves.

01

Objetos

Fundamento

Um objeto é uma coleção de pares chave: valor (também chamados de propriedade: valor). É usado para representar qualquer coisa: usuários, produtos, configurações.

Sintaxe
{ chave: valor }
Acesso por ponto
obj.chave
Acesso por colchete
obj["chave"]
Tipo
typeof → "object"
01_criando_objeto.js 
// CRIANDO UM OBJETO COM NOTAÇÃO LITERAL
// Cada "propriedade" é um par chave: valor separado por vírgula.

const pessoa = {
  nome:   "Ana Silva",     // string
  idade:  28,             // number
  ativa:  true,           // boolean
  hobbies: ["leitura", "código"], // array como valor

  // Método: uma função dentro do objeto
  apresentar: function() {
    return `Olá, sou ${this.nome} e tenho ${this.idade} anos.`;
  }
};

// ── ACESSANDO PROPRIEDADES ──
console.log(pessoa.nome);       // ponto → chave fixa
console.log(pessoa.hobbies[0]);

const chave = "idade";
console.log(pessoa[chave]);      // colchete → chave dinâmica
console.log(pessoa.apresentar());

// ── ADICIONAR / MODIFICAR / REMOVER ──
pessoa.email = "ana@email.com";
pessoa.idade = 29;
delete pessoa.ativa;

console.log(pessoa);
02_metodos_objeto.js 
// MÉTODOS ESTÁTICOS DO OBJECT
// Object.keys()   → array com as CHAVES
// Object.values() → array com os VALORES
// Object.entries()→ array de [chave, valor]

const produto = { nome: "Notebook", preco: 3500, estoque: 12 };

console.log("Chaves:", Object.keys(produto));
console.log("Valores:", Object.values(produto));
console.log("Entradas:", Object.entries(produto));

// Clonar/mesclar com spread (...)
const atualizado = { ...produto, preco: 3100, cor: "Prata" };
console.log("Atualizado:", atualizado);
console.log("Original intacto:", produto.preco);
💡 Desestruturação de objetos Forma elegante de extrair propriedades: const { nome, idade } = pessoa — cria variáveis locais com os mesmos nomes das propriedades, sem repetir pessoa..
02

for...in

Objetos → Chaves

O for...in itera sobre as chaves (propriedades enumeráveis) de um objeto. A cada volta, a variável recebe o nome de uma chave — não o valor. Para obter o valor, acesse objeto[chave].

⚠ Atenção — for...in com arrays Funciona em arrays (as chaves são os índices 0, 1, 2…), mas é não recomendado: pode iterar propriedades herdadas do protótipo e não garante a ordem. Para arrays, use for...of ou forEach (na página de Arrays).
03_for_in.js 
// FOR...IN — Itera sobre as CHAVES de um objeto
// Sintaxe: for (chave in objeto) { ... }

const carro = { marca: "Toyota", modelo: "Corolla", ano: 2022, km: 15000 };

for (const chave in carro) {
  console.log(`${chave}: ${carro[chave]}`); // valor via colchete
}

console.log("---");

// Filtrando por tipo do valor
const config = { host: "localhost", porta: 3000, debug: true };
for (const prop in config) {
  if (typeof config[prop] === "string") {
    console.log(`[string] ${prop} = "${config[prop]}"`);
  }
}

console.log("---");

// hasOwnProperty: garante que a chave é do próprio objeto
for (const prop in config) {
  if (config.hasOwnProperty(prop)) console.log("Própria:", prop);
}
03

Exercícios

Pratique aqui
A Inspecionando um Objeto ● Fácil

Use for...in para imprimir cada propriedade do objeto livro no formato "propriedade: valor". Depois imprima quantas propriedades ele tem.

  • Use for...in para iterar
  • Use Object.keys(livro).length para contar
Seu código
solução_A.js
for (const prop in livro) {
  console.log(`${prop}: ${livro[prop]}`);
}
console.log("Total de propriedades:", Object.keys(livro).length);
B Inventário de Loja (Desafio) ■ Difícil

Dado o objeto estoque, use for...in para gerar um relatório:

  • Liste cada produto com quantidade e preço
  • Calcule o valor total de cada produto (qtd × preco)
  • Marque os produtos com estoque crítico (qtd < 5)
  • Some o valor total do inventário
Seu código
solução_B.js
let total = 0;
for (const produto in estoque) {
  const { qtd, preco } = estoque[produto];
  const valor = qtd * preco;
  total += valor;
  const alerta = qtd < 5 ? " ⚠ CRÍTICO" : "";
  console.log(`${produto.padEnd(10)} qtd:${qtd} → R$ ${valor.toFixed(2)}${alerta}`);
}
console.log(`Total do inventário: R$ ${total.toFixed(2)}`);
