01

O que é Ordenação?

Fundamentos

Ordenar significa reorganizar uma sequência de elementos de acordo com uma relação de ordem (crescente, decrescente, alfabética...). É uma das operações mais frequentes em computação: buscas binárias, exibição de resultados, joins em banco de dados — todos dependem de dados ordenados.

📌 Por que começar pelo O(n²)? Bubble Sort e Selection Sort são lentos para grandes volumes, mas ensinam os fundamentos: laços aninhados, trocas, passes, e por que cada comparação importa. Eles são a base conceitual para entender Quick Sort, Merge Sort e Heap Sort.
02

Bubble Sort

O(n²) · Estável · In-place

Em cada passagem, percorremos o array comparando pares adjacentes. Se o da esquerda for maior, eles trocam de lugar — o maior vai "borbulhando" até o final. Depois de k passagens, os k maiores já estão no lugar certo e não precisam ser verificados novamente.

Experimente o visualizador: clique em Passo para avançar um a um e ver exatamente qual comparação está sendo feita, ou em Auto para assistir a ordenação completa. As barras amarelas são os elementos sendo comparados; as rosas indicam uma troca; as verdes já estão no lugar definitivo.

Implementação
bubble-sort.js 
function bubbleSort(arr) {
  const n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    // Após cada passagem i, os últimos i elementos já estão no lugar
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        // Troca: borbulha o maior para a direita
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }

  return arr;
}

// Com contagem de operações (versão didática)
function bubbleSortDetalhado(arr) {
  const n = arr.length;
  let comparacoes = 0, trocas = 0;

  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      comparacoes++;
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
        trocas++;
      }
    }
  }

  return { arr, comparacoes, trocas };
}

console.log("Ordenado:", bubbleSort([...original]));

03

Selection Sort

O(n²) · Instável · In-place

A ideia é diferente: em vez de borbulhar o maior, procuramos o menor na parte não ordenada e o movemos para o início. Após i passagens, os primeiros i elementos estão definitivamente no lugar.

Selection Sort — uma passagem
[11, 34, 25, 12, 22, 64, 90] → mínimo encontrado: 11 → troca com posição 0
[11, 12, 25, 34, 22, 64, 90] → mínimo encontrado: 12 → troca com posição 1
[11, 12, 22, 34, 25, 64, 90] → mínimo encontrado: 22 → troca com posição 2
... e assim por diante
selection-sort.js 
function selectionSort(arr) {
  const n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    // Encontra o índice do menor elemento na parte não ordenada
    let minIdx = i;
    for (let j = i + 1; j < n; j++) {
      if (arr[j] < arr[minIdx]) minIdx = j;
    }

    // Troca o menor encontrado com o primeiro da parte não ordenada
    if (minIdx !== i) {
      [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
    }
  }

  return arr;
}

const original = [64, 34, 25, 12, 22, 11, 90];
console.log("Selection Sort:", selectionSort([...original]));

// Contagem de comparações — sempre O(n²), independente da entrada
function contarComparacoes(n) {
  return (n * (n - 1)) / 2; // fórmula exata para selection sort
}
console.log(`Comparações para n=7: ${contarComparacoes(7)}`);
console.log(`Comparações para n=100: ${contarComparacoes(100)}`);

⚠ Selection vs. Bubble — diferença chave O Selection Sort faz menos trocas (no máximo n−1), enquanto o Bubble pode fazer O(n²) trocas. Isso importa quando a troca é uma operação custosa. Porém, o Selection não é adaptativo: ele faz o mesmo número de comparações mesmo se o array já estiver ordenado.
04

Comparativo de Algoritmos

Quando usar cada um

Os algoritmos O(n²) são práticos para arrays pequenos (n < 50) ou quase ordenados. Para volumes maiores, algoritmos O(n log n) são essenciais.

AlgoritmoMelhorMédioPiorTrocasEstávelAdaptativo
Bubble Sort O(n) O(n²) O(n²) O(n²) Sim Sim*
Selection Sort O(n²) O(n²) O(n²) O(n) Não Não
Insertion Sort O(n) O(n²) O(n²) O(n²) Sim Sim
Merge Sort O(n log n) O(n log n) O(n log n) O(n log n) Sim Não
Quick Sort O(n log n) O(n log n) O(n²) O(log n) Não Não

* Bubble Sort é adaptativo apenas com a otimização da flag de troca.

Exercícios

AImplementar Bubble Sort ◆ Fácil

Implemente a função bubbleSort(arr) do zero, sem olhar o exemplo acima.

  • Dois laços aninhados: externo de 0 a n-2, interno de 0 a n-2-i
  • Compare arr[j] e arr[j+1]; troque se necessário
  • Retorne o array ordenado (pode modificar in-place)
Seu código
solução_A.js
function bubbleSort(arr) {
  const n = arr.length;
  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
  return arr;
}
BBubble Sort com Flag de Otimização ◆ Médio

Adicione uma flag trocou que verifica se alguma troca ocorreu em cada passagem. Se nenhuma troca acontecer, o array já está ordenado — interrompa o laço externo mais cedo.

  • Declare let trocou no início de cada passagem
  • Após a passagem interna, se !trocou, use break
  • Teste com [1, 2, 3, 4, 5] — deveria fazer apenas 1 passagem (0 trocas)
Seu código
solução_B.js
function bubbleSortOtimizado(arr) {
  const n = arr.length;
  let passagens = 0;

  for (let i = 0; i < n - 1; i++) {
    passagens++;
    let trocou = false;

    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
        trocou = true;
      }
    }

    if (!trocou) break; // já está ordenado
  }

  console.log(`Passagens realizadas: ${passagens}`);
  return arr;
}
CImplementar Selection Sort ◆ Médio

Implemente o Selection Sort: em cada passagem, encontre o menor elemento na parte não ordenada e troque-o com o elemento na posição i.

  • Laço externo: i de 0 a n-2 (posição de destino)
  • Laço interno: encontre minIdx entre i+1 e n-1
  • Troque arr[i] com arr[minIdx] apenas se minIdx !== i
  • Compare o número de trocas com o Bubble Sort — deve ser bem menor
Seu código
solução_C.js
function selectionSort(arr) {
  const n = arr.length;
  let trocas = 0;

  for (let i = 0; i < n - 1; i++) {
    let minIdx = i;

    for (let j = i + 1; j < n; j++) {
      if (arr[j] < arr[minIdx]) minIdx = j;
    }

    if (minIdx !== i) {
      [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
      trocas++;
    }
  }

  console.log(`Trocas realizadas: ${trocas}`);
  return arr;
}
